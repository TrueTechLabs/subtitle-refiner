## 简介

Subtitle Refiner 是一款专业的字幕校对工具，专为视频创作者、翻译人员和内容制作团队设计。它可以自动优化 SRT 字幕文件，去除口语化表达，标准化技术术语，让您的字幕更加专业和易读。

### 为什么选择 Subtitle Refiner?

- **智能去口语化** - 自动识别并删除嗯、啊、那个等口语词
- **术语标准化** - 统一技术术语表达
- **保持时间戳** - 完美保留原有时间轴
- **详细报告** - 生成完整的优化概要，清晰展示修改
- **即用即走** - 安装后直接在聊天中使用，无需配置

---

## 功能特性

| 功能 | 描述 |
|------|------|
| 口语词过滤 | 智能删除嗯、啊、那个、就是、然后、呃等口语词 |
| 术语修正 | 自动将不规范术语替换为标准表达（如 "大模型" → "大型语言模型"） |
| 格式保留 | 完美保持 SRT 时间戳和索引格式 |
| 优化报告 | 生成详细的修改统计和前后对比 |
| 文件管理 | 自动保存原始文件和优化后文件 |

## 默认配置

### 口语词列表
嗯、啊、那个、就是、然后、呃

### 术语映射（语音识别纠错）

自动修正剪映等字幕软件常见的语音识别错误：

| 识别错误 | 正确词汇 | 说明 |
|----------|----------|------|
| GBT / GPTT | GPT | ChatGPT 常被识别为 GBT |
| LLM / LM | LLM | 大型语言模型缩写 |
| rag / RAGG | RAG | 检索增强生成 |

**示例效果：**
```
原始：GBT模型和rag技术的结合
纠正：GPT模型和RAG技术的结合
```

## 文件存储

- **优化字幕**：保存于 `openclaw/workspace/subtitle_refine/` 目录，命名格式为 `源文件名_优化+时间戳.srt`
